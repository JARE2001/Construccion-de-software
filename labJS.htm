¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  1. getSeconds() Returns the seconds (from 0-59)
  2. toLocaleTimeString() Returns the time portion of a Date object as a string, using locale conventions
  3. setDate() Sets the day of the month of a date object
  4. getUTCDate() Returns the day of the month, according to universal time (from 1-31)
  5. now() Returns the number of milliseconds since midnight Jan 1, 1970

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  1. concat() Joins two or more arrays, and returns a copy of the joined arrays
  2. forEach() Calls a function for each array element
  3. map() Creates a new array with the result of calling a function for each array element
  4. pop() Removes the last element of an array, and returns that element
  5. copyWithin() Copies array elements within the array, to and from specified positions
